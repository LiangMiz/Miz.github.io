Hexo入门

2020-09-20

Hexo目录结构

_config.yml

该文件可配置Hexo参数

详见 官方文档 。

scaffolds

存放生成md文章的模板

[layout] 填写该文件夹中的模板名称， 不填时默认为默认使用 _config.yml 中的 default_layout 参数代替

source

存放md源文件，主题将根据此文件夹中的md文件渲染页面

themes

存放主题文件，内有默认主题 landscape。

如需其他主题，可在 官方主题网站 下载，将主题文件夹复制在此处文件夹下。

Hexo 常用命令

hexo初始化 init

命令将在该文件夹下生成hexo目录结构

生成文章 new

命令将在 (source) 文件夹按 layour 模板新建md文件。

生成文件 generate

使用主题渲染md文件，在(public)文件夹中生成静态文件。

清空缓存 clean

命令将清除缓存文件 (db.json) 和已生成的静态文件 (public)。

本地预览 server

开启本地服务器预览网站。

部署文件 deploy

将之前生成的静态文件部署到 github 上。
